## 🤖 Prediction Model
 
A **logistic regression model** trained on 2021–2024 NRL match data.
 
### Features Used
 
| Feature | Description |
|---|---|
| `diff_trend` | Exponentially weighted points differential (last 8 games) |
| `defence_diff` | Points conceded trend differential |
| `win_rate_diff` | Weighted win rate differential |
| `margin_diff` | Average winning margin differential |
| `away_wr_diff` | Away win rate differential |
| `attack_trend_diff` | Recent scoring trend vs season average |
| `h2h_rate` | Head-to-head win rate (last 6 meetings, recency weighted) |
| `form_x_margin` | Interaction term: form × points margin |
 
### Performance
 
- **Train accuracy (2021–2024):** 68.9%
- **Test accuracy (2025 season):** 62.4% across 213 matches
- **Overfit gap:** 6.5% ✅
> 62.4% accuracy is competitive with bookmaker-implied accuracy (~63%). The model is most useful for identifying where our probability estimates diverge from Sportsbet's implied odds — these are flagged as **value bets**.

## 🏅 SuperCoach Draft Tips Methodology
 
Tips are based on two factors:
 
1. **2025 season average** — each player's established scoring baseline
2. **2026 defensive matchup** — opponent's average points conceded per game (percentile-ranked)
### Matchup Ratings
 
| Rating | Threshold | Meaning |
|---|---|---|
| 🔥 Dream matchup | Top 10% weakest defence | Ideal — target these players |
| ✅ Good matchup | Top 25% weakest defence | Solid pick |
| ➡️ Neutral | Around median | No strong advantage |
| ⚠️ Tough matchup | Bottom 25% | Proceed with caution |
| 🚫 Avoid | Bottom 10% strongest defence | Look elsewhere |
| 😴 BYE | No game this round | Don't pick |
 
> Ratings use **percentile-based thresholds** recalculated each week so ratings always reflect the current spread of defensive performance, not fixed point thresholds.
 
> **Note:** Matchup ratings reflect overall team defensive strength, not position-specific matchups. Always check Thursday team lists before locking in picks.

## ⚙️ Setup & Usage
 
### Prerequisites
 
- Python 3.9+
- Jupyter Notebook
- Free API key from [the-odds-api.com](https://the-odds-api.com)
### Installation
 
```bash
pip install requests beautifulsoup4 pandas scikit-learn xgboost
```
 
### First-time model training
 
Before running Cell 2 for the first time, train and save the model by running the training notebook. This saves `nrl_model.pkl` in your working directory.
 
### Weekly Workflow
 
Every Thursday after team lists drop:
 
1. Open your Jupyter notebook
2. **Run Cell 1** — loads all libraries and functions
3. **Run Cell 2** — fetches all data, generates predictions, saves `index.html`
4. Push `index.html` to your GitHub repo
5. Your newsletter is live ✅
```
Total time: ~3 minutes
API requests used: 1 (Odds API — 499 remaining on free tier)
```
